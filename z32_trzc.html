Model pudełkowy w CSS

Pytania teoretyczne

  1. Podaj definicję modelu pudełkowego

    Model pudełkowy (ang. box model) to sposób reprezentacji elementów HTML jako prostokątnych pudełek. Każdy element składa się z czterech warstw: zawartości (content), wewnętrznego marginesu (padding), obramowania (border) i zewnętrznego marginesu (margin). Model ten umożliwia kontrolowanie odstępów i wyglądu elementów na stronie.

Uwagi o modelu pudełkowym

  • Rozmiar elementu HTML zależy od sumy: szerokości contentu, paddingu, borderu i marginesu (chyba że zastosujemy box-sizing: border-box).
  • Ustawienie właściwości CSS takich jak margin lub padding wpływa na układ i przestrzeń wokół elementów.

Ilustracja modelu pudełkowego

Tabela – Elementy modelu pudełkowego

Element Opis
Content Zawartość elementu, np. tekst lub obrazek. To główna część widoczna dla użytkownika.
Padding Odstęp między zawartością a obramowaniem (borderem). Przezroczysty obszar wokół contentu.
Border Ramka otaczająca padding i content. Może mieć różną grubość, kolor i styl.
Margin Odstęp między borderem a innymi elementami strony. Przezroczysty obszar na zewnątrz elementu.
